## 📦 Installation Guide

### Step 1: Get Your Widget Code
1. Log in to your [Givelink Dashboard](https://givelink.app)
2. Navigate to "Charities" → "Widget Generator"
3. Select the nonprofit you want to promote
4. Click "Copy Code"
5. Your embed code is now ready to use!

### Step 2: Choose Your Platform & Follow the Guide

- **Using WordPress?** → [WordPress Integration Guide](en/iframe/wordpress.md)
- **Using Squarespace?** → [Squarespace Integration Guide](en/iframe/squarespace.md)
- **Using Wix?** → [Wix Integration Guide](en/iframe/wix.md)
- **Using a custom website?** → [Native JavaScript Guide](en/iframe/native.md)

### Step 3: Add Your Donation Button

Each platform has a slightly different process. Refer to your platform's specific guide above.

### Step 4: Customize (Optional)

Customize the button's:
- Color and styling
- Text and call-to-action
- Size and positioning
- Hover effects

## ❓ Frequently Asked Questions

### General Questions

**Q: Do I need to be a developer to use Givelink?**  
A: No! WordPress, Squarespace, and Wix users don't need any coding knowledge. Just follow the step-by-step guides.

**Q: Is the widget free to use?**  
A: The widget itself is free! Givelink takes a small transaction fee on donations.

**Q: Can I customize the button's appearance?**  
A: Yes! You can customize colors, text, size, and styling to match your brand.

**Q: Will it work on my website?**  
A: Givelink supports WordPress, Squarespace, Wix, and any custom website. If you're using a different platform, try the Native JavaScript integration.

**Q: Can I have multiple donation buttons on the same page?**  
A: Yes! Simply add the button code multiple times on the same page.

**Q: Which platform is easiest to set up?**  
A: WordPress and Squarespace are equally easy (5 minutes). Wix takes 10 minutes due to Dev Mode setup.

### Technical Questions

**Q: Is my donor data secure?**  
A: Yes! All payments are PCI-compliant and securely processed. See our [Security Policy](SECURITY.md).

**Q: Can I track donations?**  
A: Yes! Use your Givelink Dashboard to view analytics, donor information, and donation history.

**Q: What payment methods are supported?**  
A: Credit cards, PayPal, Apple Pay, Google Pay, and more (varies by region).

**Q: Will the widget work on mobile?**  
A: Absolutely! The widget is fully responsive and mobile-optimized.

### Troubleshooting Questions

**Q: The widget isn't showing on my page. What do I do?**  
A: Check our [Troubleshooting Guide](en/iframe/troubleshooting.md) for platform-specific solutions.

**Q: I'm seeing JavaScript errors. How do I fix this?**  
A: See the [Troubleshooting Guide](en/iframe/troubleshooting.md) for debugging steps.

**Q: Still stuck?**  
A: Contact support at contact@givelink.app or [create an issue on GitHub](https://github.com/givelink/documentation/issues).

## 🎯 Best Practices

### Button Placement

✅ **Best Locations:**
- In the header/navigation bar
- At the end of your mission statement
- In the sidebar
- On a dedicated "Support Us" page
- In blog posts (below the content)

### Button Text & Messaging

✅ **Good Examples:**
- "Donate Now"
- "Give In-Kind"
- "View Our Needs"
- "Support [Organization Name]"
- "Make an Impact"

### Mobile Optimization

- Ensure button is thumb-friendly (minimum 44x44px)
- Test on phones and tablets
- Use responsive sizing

### Donation Messaging

✅ **Effective Messaging:**
- Explain what donations support
- Share impact stories
- Show exactly where money goes
- Use specific dollar amounts for context

## 🆘 Troubleshooting

### Quick Fixes

**Widget not displaying?**
- Verify you copied the complete code from Givelink
- Check browser console for JavaScript errors
- Clear browser cache and reload

**Button not working?**
- Ensure the button has the correct class or onclick handler
- Verify the widget script is loaded first

**Styling conflicts?**
- Inspect the button using browser DevTools
- Check CSS specificity
