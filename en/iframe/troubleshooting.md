# Troubleshooting

---

## Common Mistakes

**The widget isn't showing on my page**
- Make sure you copied the full embed code from your [Givelink Dashboard](https://givelink.app/en/charities/widget-gen)
- Double-check that the code was pasted in the right place (see your platform guide)
- Try refreshing the page (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)

**The widget shows but donations aren't going through**
- Make sure you selected the correct nonprofit when generating the code — the widget is nonprofit-specific
- Try clicking the donation button yourself to test the full flow
- Try opening the widget in a different browser to rule out a browser-specific issue

**I made a change but don't see it on my site**
- Refresh your page (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)
- Wait a few minutes — some platforms take time to publish changes

**The page looks broken after adding the widget**
- Remove the embed code and re-paste it — a copy/paste error may have cut off part of the code
- Make sure you didn't accidentally paste the code inside another HTML element
