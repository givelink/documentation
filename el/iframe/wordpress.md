# Ενσωμάτωση WordPress

Ακολουθήστε [αυτό](https://youtu.be/_UqKMOnC1_0) το βίντεο βήμα προς βήμα, ή διαβάστε τα παρακάτω βήματα:

1. Αντιγράψτε τον κωδικό HTML <strong>επιλέγοντας</strong> τον μη κερδοσκοπικό οργανισμό που θέλετε να προωθήσετε από τη [Σελίδα Αντιγραφής Widget](https://givelink.app/charities/widget-gen)

   ![Αντιγραφή Κωδικού](/assets/widget-assets-greek.png)

2. Μεταβείτε στον πίνακα διαχείρισης WordPress και συνδεθείτε.

3. Πλοηγηθείτε στη σελίδα ή ανάρτηση όπου θέλετε να προσθέσετε το κουμπί δωρεάς.

4. Προσθέστε ένα νέο μπλοκ Custom HTML κάνοντας κλικ στο κουμπί "+" και επιλέγοντας "Custom HTML" από τις επιλογές μπλοκ.

![Προσθήκη Μπλοκ Custom HTML](/assets/wp/wp-code-insert.png)

5. Επικολλήστε μέσα στο μπλοκ κώδικα το script που παρέχεται από το βήμα 1.
   (αυτό δεν θα εμφανίζεται στους χρήστες)

![Επικόλληση Κωδικού](/assets/wp/wp-code-paste.png)

6. Προσθέστε ένα νέο μπλοκ κουμπιού κάνοντας κλικ στο κουμπί "+" και επιλέγοντας "Buttons" από τις επιλογές μπλοκ.

![Προσθήκη Μπλοκ Κουμπιού](/assets/wp/wp-button-insert.png)

7. Τοποθετήστε το κουμπί στη σελίδα. Μπορείτε ελεύθερα να προσαρμόσετε χρώματα και γραμματοσειρά σύμφωνα με τον ιστότοπό σας. Βεβαιωθείτε ότι η Παρότρυνση για Δράση είναι <strong>"Δώρισε σε είδος"</strong>, <strong>"Δες τις ανάγκες μας"</strong> ή κάτι παρόμοιο.

8. Με επιλεγμένο το κουμπί, στη δεξιά πλευρά, κάντε κλικ στο "Advanced" και προσθέστε στο πεδίο Additional css class(es) το εξής: `givelink-button`.

![Προσθήκη CSS Κλάσης](/assets/wp/wp-button-classname.png)

9. Αποθηκεύστε τις αλλαγές και δημοσιεύστε όταν είστε έτοιμοι.
