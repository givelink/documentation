## 🚀 Γρήγορη Εκκίνηση

### Βήμα 1: Αποκτήστε τον Κωδικό του Widget
1. Συνδεθείτε στο [Givelink Dashboard](https://givelink.app)
2. Μεταβείτε στο "Charities" → "Widget Generator"
3. Επιλέξτε τον μη κερδοσκοπικό οργανισμό που θέλετε να προωθήσετε
4. Κάντε κλικ στο "Copy Code"
5. Ο κωδικός ενσωμάτωσής σας είναι έτοιμος για χρήση!

## 🆘 Αντιμετώπιση Προβλημάτων

**Το widget δεν εμφανίζεται;**
- Επαληθεύστε ότι αντιγράψατε τον πλήρη κώδικα από το Givelink
- Ελέγξτε την κονσόλα του προγράμματος περιήγησης για σφάλματα JavaScript
- Εκκαθαρίστε την προσωρινή μνήμη του προγράμματος περιήγησης και ανανεώστε

**Το κουμπί δεν λειτουργεί;**
- Βεβαιωθείτε ότι το κουμπί έχει τη σωστή κλάση ή χειριστή onclick
- Επαληθεύστε ότι το script του widget φορτώθηκε πρώτα
