# Αντιμετώπιση Προβλημάτων

---

## Συνηθισμένα Λάθη

**Το widget δεν εμφανίζεται στη σελίδα μου**
- Βεβαιωθείτε ότι αντιγράψατε τον πλήρη κώδικα ενσωμάτωσης από το [Givelink Dashboard](https://givelink.app/en/charities/widget-gen)
- Ελέγξτε διπλά ότι ο κώδικας επικολλήθηκε στο σωστό σημείο (δείτε τον οδηγό πλατφόρμας σας)
- Δοκιμάστε να ανανεώσετε τη σελίδα (Ctrl+Shift+R στα Windows, Cmd+Shift+R στο Mac)

**Το widget εμφανίζεται αλλά οι δωρεές δεν ολοκληρώνονται**
- Βεβαιωθείτε ότι επιλέξατε τον σωστό μη κερδοσκοπικό οργανισμό κατά τη δημιουργία του κώδικα — το widget είναι ειδικό για κάθε οργανισμό
- Δοκιμάστε να κάνετε κλικ στο κουμπί δωρεάς ο ίδιος για να ελέγξετε τη ροή
- Δοκιμάστε να ανοίξετε το widget σε διαφορετικό browser για να αποκλείσετε πρόβλημα συγκεκριμένου browser

**Έκανα μια αλλαγή αλλά δεν την βλέπω στον ιστότοπό μου**
- Ανανεώστε τη σελίδα (Ctrl+Shift+R στα Windows, Cmd+Shift+R στο Mac)
- Περιμένετε λίγα λεπτά — ορισμένες πλατφόρμες χρειάζονται χρόνο για να δημοσιεύσουν τις αλλαγές

**Η σελίδα φαίνεται χαλασμένη μετά την προσθήκη του widget**
- Αφαιρέστε τον κώδικα ενσωμάτωσης και επικολλήστε τον ξανά — ένα σφάλμα κατά την αντιγραφή/επικόλληση μπορεί να έχει κόψει μέρος του κώδικα
- Βεβαιωθείτε ότι δεν επικολλήσατε κατά λάθος τον κώδικα μέσα σε άλλο στοιχείο HTML
